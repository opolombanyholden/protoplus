Shift
Sets animatically any numeric style value of an element. Also sets colors and scroll amounts
it a generic animation library.
$(element).shift({height:100, width:200, duration:3});
Option Default Values Description
duration 1 numeric value as seconds Defines how much time to it will take to complete the animation.
easing sineOut Transition name Defines which easing method will be applied for the animation.
onStart - Function A function which will be run before the animation starts.
onEnd - Function A function which will be run after the animation finishes.
remove false Boolean If true, removes the element from dom when the animation finishes.
Style Property - Any Numeric Value Accepts an style propety of an HTML property with a numeric value.
Such as height, width, borderWidh, fontSize, opacity, etc.
properties must be camelized 'font-size to fontSize',
this option also accepts color values such as background, borderColor, color.
scrollX or scrollLeft - Any positive numeric value Horizontally scrolls the overflowed elements
scrollY or scrollTop - Any positive numeric value Vertically scrolls the overflowed elements
